Mengapa PHP:
1. relatif mudah untuk pemula
2. pre-requisite (HTML+CSS)
3. gratis dan open sourse
4. dukungan komunitas sangatlah besar Contoh: stackoverflow
5. dokumentasi sangat lengkap (http://php.net/manual/en/)
6. PHP dibuat khusus untuk web
7. kebanyakan website yang ada saat ini dibuat dengan PHP (https://w3tchs.com)
8. banyak website besar dibuat dengan PHP (facebook, Yahoo!, Wikipedia, flickr dll)
9. Content Management System yang menggunakan PHP (WordPress, Joomla!, Drupal, moodle dll)
10. Framework PHP (Codeigniter, CakePHP, yiiframwork, laravel dan Symfony)

7. studi kasus website sederhana
8. database

12. ajax
